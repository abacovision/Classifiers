# Classifiers

### Why this repo exists?

The function of this repo, is to share the algorthims of image classifiers. The essence is to present the mathematical base of them, by this way understanding the algorithm from it's most acurate source.

### I can use the algorithms presented here?

Yes! This is the idea, when a i learn something new, i will develop the new algorithm and post here. The next step is you clonning this repo and trying the algorithm by youself.

### How i can contribute with this repo?

Just send a commit request or use the algorithm and share the link in the community, you will aways be welcome to contribute.
